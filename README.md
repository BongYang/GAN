# Pytorch-lightning GAN

### Version
```
pytorch 1.11.0
pytorch-lightning 0.8.5
```

### Execution
```
python train.py --model_name "GAN"
```

### GANs
1. Basic GAN
2. Conditional GAN
3. Deep Convolutional GAN
4. Least squares GAN
5. Pix2Pix - Image-to-Image Translation
